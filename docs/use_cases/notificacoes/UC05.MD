### Especificação de Caso de Uso: Projeto Legixtech

#### UC05 – Configurar e Receber Notificações

* **Ator Principal:** Servidora Pública (Carla Figueiredo).
* **Objetivo:** Receber alertas rápidos sobre o andamento de leis relevantes sem precisar entrar no aplicativo diariamente.
* **Pré-condição:** Usuária autenticada no aplicativo.
* **Fluxo Principal:** 
    1. A usuária acessa o ícone de "Perfil" e navega até "Configurações de Notificação". 
    2. O sistema exibe opções de personalização de alertas (ex: Projetos Salvos, Temas de Interesse, Resumo Diário). 
    3. A usuária ativa o botão (toggle) para receber "Notificações Push" sobre os Projetos Salvos. 
    4. O sistema salva as preferências automaticamente e solicita permissão do sistema operacional para enviar notificações (se for o primeiro acesso). 
    5. Quando um projeto salvo sofre alteração no Congresso (ex: aprovado em comissão), o sistema dispara uma notificação push. 
    6. A usuária visualiza a notificação na tela de bloqueio do celular com um texto curto e direto (ex: "O PL 123/2026 avançou para votação!"). 
    7. A usuária toca na notificação e o sistema abre diretamente a tela do projeto correspondente.
* **Fluxo Alternativo (Desativar Alertas):** 
    1. A usuária se sente sobrecarregada com os alertas, acessa as configurações e desativa o toggle de um tema específico. 
    2. O sistema cessa imediatamente o envio de notificações daquele tema, mantendo os alertas no "Histórico de Notificações" interno do app.
* **Pós-condição:** A usuária tem controle total sobre quais alertas recebe e se mantém informada de forma passiva.
