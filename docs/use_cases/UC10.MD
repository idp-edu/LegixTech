### Especificação de Caso de Uso: Projeto Legixtech

#### UC10 – Consultar o Chatbot para Dúvidas
* **Ator Principal:** Estudante Universitário (Lucas Mendes).
* **Objetivo:** Tirar dúvidas sobre o funcionamento de uma lei ou traduzir jargões técnicos em tempo real.
* **Pré-condição:** Usuário logado e lendo uma proposição legislativa.
* **Fluxo Principal:**
    1. O usuário encontra o termo técnico "Veto Presidencial com Derrubada" no andamento de um projeto. 
    2. O usuário toca no botão flutuante de Chatbot no canto inferior direito da tela. 
    3. O sistema abre a interface de chat sobrepondo parcialmente a tela. 
    4. O sistema exibe uma mensagem de boas-vindas e oferece botões rápidos com perguntas frequentes baseadas no projeto que o usuário está lendo (ex: "O que significa este status?"). 
    5. O usuário prefere digitar e envia: "O que é derrubada de veto?". 
    6. O sistema processa a pergunta e responde em segundos usando linguagem informal e analógica (ex: "É quando o Congresso decide ignorar o 'não' do Presidente e aprovar a lei mesmo assim..."). 
    7. O usuário lê a resposta, avalia como "Útil" (ícone de joinha) e fecha o chat.
* **Fluxo Alternativo (O Chatbot não entende a pergunta):** 
    1. O usuário faz uma pergunta muito complexa ou fora do escopo. 
    2. O sistema responde com uma mensagem padrão: "Desculpe, ainda estou aprendendo. Você quis dizer X ou Y?" ou fornece um link para o glossário oficial da Câmara.
* **Pós-condição:** A dúvida técnica é resolvida sem que o usuário precise sair do aplicativo e buscar no Google.
