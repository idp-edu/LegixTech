### Especificação de Caso de Uso: Projeto Legixtech

#### UC08 – Utilizar o Aplicativo Fácil / Onboarding

* **Ator Principal:** Aposentada (Dona Maria Lúcia).
* **Objetivo:** Aprender a usar as principais funções do aplicativo logo no primeiro acesso para não depender de ajuda de terceiros.
* **Pré-condição:** O aplicativo foi instalado e aberto pela primeira vez.
* **Fluxo Principal:** 
    1. Após a tela de login (UC01), o sistema inicia automaticamente o "Tutorial Interativo". 
    2. A tela é escurecida, e um foco de luz (highlight) destaca a Barra de Busca, acompanhado de um balão explicativo em fonte grande: "Comece pesquisando aqui". 
    3. A usuária toca no botão "Entendi" no balão explicativo. 
    4. O sistema move o foco para o ícone de Favoritos, explicando sua função. 
    5. A usuária conclui o fluxo de 3 passos rápidos. 
    6. O sistema exibe uma mensagem de sucesso: "Você está pronta! Se precisar, o tutorial está disponível no seu Perfil."
* **Fluxo Alternativo (Pular Tutorial):**
    1. Durante a exibição do primeiro balão explicativo, a usuária visualiza o botão "Pular Tutorial" no canto da tela. 
    2. A usuária toca em "Pular", e o sistema encerra o onboarding, levando-a diretamente para a tela inicial.
* **Pós-condição:** A usuária está capacitada para navegar pelo aplicativo ou sabe onde buscar as instruções novamente se tiver dúvidas.
