### Especificação de Caso de Uso: Projeto Legixtech

#### UC02 – Salvar Projetos na Lista de Monitoramento

* **Ator Principal:** Servidora Pública (Carla Figueiredo).
* **Objetivo:** Salvar projetos específicos para não perdê-los de vista em meio à vasta produção legislativa.
* **Pré-condição:** Usuária lendo a página de detalhes de um Projeto de Lei.
* **Fluxo Principal:** 
    1. A usuária analisa um projeto pertinente à sua área de trabalho. 
    2. A usuária toca em um botão grande e visível com o ícone de marcador (bookmark) e o texto "Salvar para Acompanhar". 
    3. O sistema muda a cor do botão para confirmar a ação e exibe uma notificação verde no topo: "Projeto salvo no seu Perfil". 
    4. A usuária acessa o menu inferior e toca em "Meu Perfil". 
    5. A usuária abre a seção "Projetos Salvos". 
    6. O sistema carrega a lista com todos os projetos salvos, mostrando o status mais recente de cada um.
* **Fluxo Alternativo (Lista de Salvos Vazia):** 
    1. A usuária entra na seção "Projetos Salvos" sem ter salvo nada antes. 
    2. O sistema exibe uma tela com uma ilustração (empty state) e o texto: "Você ainda não salvou nenhum projeto. Navegue pelos temas e clique em 'Salvar' para acompanhar leis importantes para você."
* **Pós-condição:** Projetos de alto valor para a usuária ficam centralizados em um ambiente de fácil acesso futuro.
